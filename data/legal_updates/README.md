# Legal Updates Tool

## Что делает
Инструмент `src/tools/legal_updates_fetcher.py` обновляет юридическую базу из официальных источников и формирует отчет для юриста.

## Где что хранится
- Источники: `data/legal_updates/sources.json`
- Состояние дедупликации: `data/legal_updates/state.json`
- Отчеты: `obsidian_vault/01_Law/Updates/`
- Последний отчет: `obsidian_vault/01_Law/Updates/LATEST_UPDATES.md`

## Запуск
```bash
python3 src/tools/legal_updates_fetcher.py
```

## Рекомендация по расписанию
Ежедневно в 08:30 и 17:30 локального времени.

## Важно
- В отчеты попадают только найденные публикации с URL.
- Если источник недоступен, это фиксируется в блоке ошибок.
- Инструмент не делает юридических выводов, только сбор и журналирование публикаций.
